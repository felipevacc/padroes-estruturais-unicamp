Este projeto apresenta a refatoração de um sistema legado de cobrança corporativa utilizando padrões estruturais de projeto.

O foco da atividade foi identificar problemas arquiteturais existentes na implementação original, especialmente relacionados à integração com gateways de pagamento externos de interfaces incompatíveis e à combinação de ajustes por meio de parâmetros booleanos. A partir dessa análise, foram aplicados padrões estruturais para melhorar a organização, a flexibilidade e a manutenibilidade do sistema, preservando sua funcionalidade original.
